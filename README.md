# 📊 Análise de Clustering com Machine Learning

## 📌 Sobre o projeto
Este projeto aplica técnicas de clustering para identificar padrões em um conjunto de dados, utilizando algoritmos de aprendizado não supervisionado.

## 🧠 Técnicas utilizadas
- K-Means
- Análise exploratória de dados
- Normalização de dados

## 🛠️ Tecnologias
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## 📈 Objetivo
Explorar técnicas de agrupamento e entender como diferentes algoritmos podem identificar padrões em dados sem rótulos.

## ▶️ Como executar
1. Abrir o notebook no Google Colab ou Jupyter
2. Executar todas as células

## 📎 Observação
Projeto desenvolvido para fins de estudo em Machine Learning.
